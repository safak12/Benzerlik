# Benzerlik

C# Windows Forms ile geliştirilmiş metin ve veri benzerliği analiz uygulamasıdır. Projede belirli alanlar üzerinde benzerlik oranı hesaplama ve çok iş parçacıklı çalışma mantığı kullanılmıştır.

## Özellikler

- Metin alanları arasında benzerlik karşılaştırma
- Farklı filtre alanlarıyla analiz yapma
- Çok iş parçacıklı çalışma desteği
- Eşik değere göre sonuç üretme

## Kullanılan Teknolojiler

- C#
- Windows Forms
- .NET Framework

## Proje Amacı

Bu proje, veri karşılaştırma, string benzerliği hesaplama ve çok iş parçacıklı programlama konularında deneyim geliştirmek amacıyla hazırlanmıştır.

## Öğrendiklerim

- Windows Forms arayüz geliştirme
- Thread mantığı ile performans yaklaşımı kurma
- Veri filtreleme ve karşılaştırma işlemleri
- Kullanıcıdan alınan parametrelerle dinamik işlem yürütme

## Geliştirme Notları

Projede masaüstü uygulama yaklaşımıyla veri analizi senaryosu ele alınmıştır. İleride sabit dosya yolları kaldırılarak daha taşınabilir hale getirilebilir.
